## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/A-Navaneetha/Portfolio.git
```

Navigate to the project directory:

```bash
cd Portfolio
```

Install dependencies:

```bash
npm install
```

Run the application:

```bash
npm start
```

Open:

```text
http://localhost:3000
```
